# 📖 DevByte – Modern Blog Application  

A full-stack blog platform built with **React.js**, **Node.js**, **Express.js**, and **MongoDB**, designed for smooth content creation, browsing, and management.

| Layer      | Technology |

| Database   | MongoDB |
| Auth       | JSON Web Token (JWT) |
